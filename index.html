Zelda Knockoff

This project was started in school where we had to demonstrate our knowledge of static objects and use a singleton method to keep values while going between different scenes. I decided that I wanted to also test my skills against programmable animations and use tools to create terrain through multiple meshes, as well as bake ambient lighting.
